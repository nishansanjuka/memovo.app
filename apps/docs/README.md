
# Memovo Documentation App

This is the documentation site for the Memovo platform. It provides API references, architecture guides, and service documentation for all Memovo microservices.

## Getting Started

To run the documentation site locally:

```bash
pnpm dev
```

Open [http://localhost:3001](http://localhost:3001) in your browser to view the docs.

## Editing Documentation

- Edit or add pages in the `pages/` directory using MDX
- Update site configuration in `zudoku.config.tsx`
- Add OpenAPI specs, architecture diagrams, and guides as needed

## Contributing
