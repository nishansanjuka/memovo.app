Welcome to the documentation for the Memovo application platform!

Memovo is a modern, full-stack microservices platform featuring:

- Spring Boot API (Java)
- NestJS Gateway (TypeScript)
- Python LLM Service
- Flutter Mobile App

Use this site to explore API docs, architecture, and development guides for the Memovo ecosystem.
